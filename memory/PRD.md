# Ronnie D Dunn - Premium Gospel Artist Website

## Project Overview
Premium, high-end artist website for gospel artist Ronnie D Dunn featuring a timeless, elegant, and spiritually grounded design with black, gold, and warm neutral color palette.

## Original Problem Statement
Create a premium, high-end artist website page for gospel artist Ronnie D Dunn. The design should feel timeless, elegant, and spiritually grounded, using a black, gold, and warm neutral color palette. The overall aesthetic is polished, professional, and reverent — luxury gospel, not flashy or trendy.

## User Personas
- **Radio Programmers**: Need professional EPK and press materials
- **Playlist Editors**: Looking for quality gospel music for curation
- **Sync Supervisors**: Seeking faith-based music for media projects
- **Church Leaders**: Interested in booking for church events
- **Gospel Audience**: Fans seeking uplifting, faith-driven music

## Core Requirements (Static)
1. Full-screen cinematic hero section with professional portrait
2. Metallic gold artist logo with light sweep animation
3. Complete artist bio highlighting faith journey
4. Music streaming integration (Spotify, Apple Music, YouTube)
5. Video embed section
6. Message & Mission statements
7. Press & Media download section
8. Professional footer with social links
9. Responsive design across all devices
10. Black, gold, and warm neutral color scheme

## Architecture & Tech Stack
- **Frontend**: React 19, Tailwind CSS, Shadcn UI components
- **Styling**: Custom CSS with gold (#d4af37) color scheme
- **Icons**: Lucide React (no emoji icons)
- **Animations**: CSS animations (logo glow, light sweep, fade-in effects)
- **Backend**: Not implemented (static website)
